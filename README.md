#  Movie Recommendation-System

A movie recommendation system that suggests movies based on user preferences. This project uses data from The Movie Database (TMDb) API.

# Technologies Used

1. Python: Programming language used for the backend logic.
2. Streamlit: Framework for creating the interactive web application.
3. TMDb API: Provides movie data and metadata.
4. Pandas: Data manipulation and analysis.
5. NumPy: Mathematical operations.
6. Scikit-Learn: Machine learning algorithms for recommendation systems.

# Configuration
TMDb API Key:

1. Sign up at TMDb and obtain an API key.
2. Replace "your_tmdb_api_key" in the configuration file with your API key.
